**About**
This project uses Selenium to automate a simple Amazon search test.
It opens Amazon, searches for “laptop,” and prints the top 5 results.

**Requirements**
Python installed
Selenium module
ChromeDriver matching your Chrome version
Install Selenium:
pip install selenium

**How to Run**
Run this command:
python amazon_search.py

**What It Does**
Opens Amazon
Searches for “laptop”
Loads results
Prints 5 product titles
Closes browser
